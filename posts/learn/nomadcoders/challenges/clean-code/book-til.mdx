### TIL (Today I Learned) 작성 방법

- [커뮤니티의 노개북 TIL 작성 페이지](https://nomadcoders.co/community/clean_code)로 오세요
- [TIL 예시](https://nomadcoders.co/community/thread/1866)를 참고하여, 같은 양식으로 그날 그날의 독서 감상평을 작성합니다.

![Untitled](https://i.ibb.co/K59Xyn4/Screen-Shot-2022-02-16-at-7-06-34-PM.png)

- 작성 후, 생성된 TIL 링크를 제출하면 끝!

![Untitled](https://i.ibb.co/y88GqPm/Screen-Shot-2022-02-15-at-5-50-47-PM.png)

- 본인 블로그, 깃허브, SNS에 기록을 쌇아도 좋습니다. 이때, #노마드코더 #북클럽 #노개북 해시태그를 꼭! 남겨주세요.
    - 예시 1. [친슈님의 블로그](https://achelous1.github.io/wiki/clean-code-2022-02-06/)
    - 예시 2. [Robinyeon님의 깃허브](https://github.com/robinyeon/TIL/blob/main/BOOKS/%ED%81%B4%EB%A6%B0%20%EC%BD%94%EB%93%9C(Clean%20Code).md)
    - 예시 3. [su_gu_daily님의 인스타](https://www.instagram.com/p/CZb9V9NvJxf/)
- 마찬가지로 생성된 해당 블로그, 깃허브, SNS의 개별 링크를 제출하면 됩니다.
- 마감시간 (새벽 6시)까지 해당 링크를 계속 변경. 업데이트 할 수 있습니다.

<br />

### ⚠️ 주의사항

- 정상적으로 제출되었다면. 저장되었다는 문구 “Saved!”가 하단에 뜹니다.
- 또한, 노마드 계정 이메일로 제출 확인 이메일이 발송됩니다.
- **이메일이 안 왔다면, 정상 제출이 되지 않은 것이니 제출 확인은 이메일이 왔는지 꼭 체크** 해주세요!
- 제출 확인 이메일이 오지 않고, 정상 제출이 되지 않는다면? help@nomadcoders.co 로 아래 내용을 적어서 이메일로 보내주세요.
    - 계정 이메일
    - 참여하는 북클럽 명
    - 과제#
    - 과제 링크 URL
    - 발생하는 에러 내용 설명 및 전체화면 스크린샷
