**애자일 소프트웨어 장인 정신**

좋은 코드란 무엇인가요? 

바로 떠오르지 않는다면 이 책의 1장이라도 읽어봅시다.

- 나쁜 코드는 서비스를 망하게 한다!
- 따로국밥 처럼 맘대로 짜는 코드를 피하고 싶다면?
- 안타깝기 그지없는 코드를 읽어본 경험이 있다면?

좋은 코드와 나쁜 코드를 구분하며 깨끗한 코드를 실천하도록 도와주는 꿀팁의 책!

<br />

### 이런 분들 오세요.

- 미루고 미루던 개발자 필독서, 이번 기회에 드디어 읽고 싶은 사람.
- 2022년에는 책 읽는 습관을 만들어보고 싶은 사람.
- 다른 사람들과 함께 읽으면서 동기부여를 받고 이해도도를 넓히고 싶은사람.
- 독서 기록을 제대로, 꾸준히 써보고 싶은 사람.
