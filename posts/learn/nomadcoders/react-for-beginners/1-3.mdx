이 코스를 수강하기 위해 필요한 것은 무엇일까요?

이상적으로는 주니어 트랙을 모두 마치는 걸 추천합니다.

하지만 혹시 시간이 없다면 가장 중요한 ‘바닐라 JS로 크롬앱 만들기’는 꼭 수강하시길 바랍니다.

React JS 코스를 수강하기 전에 반드시 끝내고 와야합니다.

왜나하면, React JS 강의에서 기본적으로 바닐라 JS에서 배운 것의 연장 선으로 React JS를 가르칠 예정입니다.

바닐라 JS 코스에서 배운 모든 것을 기억하고 있다는 가정하에 Javascript에 익숙하다는 전제하에 진행하겠습니다.

이 React JS 코스는 Javascript 입문자를 위한 코스는 아닙니다. React JS 입문자를 위한 코스입니다.

그렇기 때문에 이 강의를 듣기 전에 Javascript를 익숙하고 편하게 다룰 수 있어야 합니다.

Javascript에 익숙하지 않다면 ‘바닐라 JS로 크롬앱 만들기’ 코스는 꼭 수강하시길 바랍니다.

또한 참여할 수 있는 챌린지도 있습니다.

챌린지에 참여하면 코스를 마치는데 도움이 될 거고, 배운 것을 연습할 수 있습니다.

그러므로 ‘바닐라 JS로 크롬앱 만들기 코스’에 참여하고 졸업하고 오십시오!

<br />

다시한번 말하지만 기초는 다루지 않을 생각입니다.

event listener가 무엇인지 HTML을 Javascript에서 어떻게 수정하는지 이런 주제는 이미 바닐라 JS 코스에서 다룬 이야기이기 때문에 아무것도 다루지 않을 것입니다.

<br />

HTML, CSS, Javsscript 나중에는 Node JS에 대해서도 조금 알고 있으면 좋을 것입니다.

이제 컴퓨터에는 무엇을 설치해야할지 알아봅시다.

이 강의를 시작할 때에는 모든 것을 브라우저에서 작업할 것입니다.

구글 크롬, 브레이브, 사파리 등 원하는 어떤것이던 상관없습니다.

그리고 Visual Studio Code와 같은 텍스트 에디터가 필요합니다.

그러나 만약에 이미 바닐라 JS 코스를 마쳤다면 이미 가지고 있을 것입니다.

바닐라 JS 코스에서 공통적인 내용을 유추해내면서 진행할거고, 여기에 기반하여 여러가지를 설명하겠습니다.

<br />

그럼 필요한 것은 이게 전부 입니다. ‘바닐라 JS로 크롬앱 만들기’ 코스의 수강과 브라우저와 Visual Studio Code를 준비합시다.
