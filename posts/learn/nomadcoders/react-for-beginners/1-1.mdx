노마드코더의 ‘초보를 위한 리액트 JS’ 코스에 온 것을 환영합니다.

함께 ReactJS를 배워나갈 생각에 정말 신납니다.

프론트엔드 개발자로서 React JS를 배운것은 저에게 있어 정말 최고의 선택이었습니다.

의심할 여지 없이, React JS는 프론트엔드에 무언가를 빌드하기 위해서, 웹사이트를 interactive하게 만들고, 아름다운 것들을 제작하고 사람들이 사용하는 무언가를 만들기 위해서 제가 배운 최고의 도구이고 제가 했던 최고의 시간 투자였습니다.

저는 React JS를 2015년 초에 배웠습니다.

그 때부터 처음으로 React JS를 살펴보고 배우기 시작했습니다. 그때 React JS는 페이스북에 의해 만들어졌지만 작았고 정말 작은 난쟁이였습니다.

그때는 Angular가 정말 거대했고, 그리고 Backbone, MeteorJS, EmberJS등 프론트엔드 framework가 너무 많았을 때 입니다.

그러나 저는 작지만 작업한 방식이 너무 마음에 들었기 떄문에 React JS를 선택했습니다.

페이스북에서 만든 것이기도 하고요

저는 React JS 믿었고 그리고 거의 2022년이 다 되어가는 지금 React JS는 살아 남았고 절대적인 왕이라고 할 수 있습니다.

페이스북은 여전히 React JS를 사용하고 있고, 대부분의 거대한 웹사이트들 인스타그램, 핀터레스트, 우버, 넷플릭스 이런 사이트들도 React JS를 사용하고 있습니다.

그래서 여기에 투자한 시간이 프론트엔드 개발자로서 최고 였다고 말할 수 있습니다.

이제 여러분과 함께 해서 기쁘고 여러분께 제가 가장 좋아하는 도구를 소개해주려고 합니다.

마치 내가 가장 좋아하는 장난감을 보여주는 것과 같은 느낌입니다.

여러분들은 그 장난감을 가지고 최대한 잘 가지고 놀면 되는 겁니다.

그것이 내가 이 코스를 만든 이유이고 또한 이 코스를 업데이트하는 이유입니다.

이 강의가 React JS를 처음으로 가르치는게 아니고 벌써 같은 코스를 두분이나 만들었었습니다.

이번 강의가 업데이트하는 세 번째 강의 입니다.

이 강의에 처음 버전에서는 React JS가 왜 어썸한지 설명해서 저를 믿어 달라고 설득하기 위해 파워포인트도 만들었습니다.

지금은 React JS가 심지어 더 좋아졌기 때문에 다음 비디오에서 몇가지 숫자를 보여드리겠습니다.

그걸 보면 React JS가 얼마나 크고 대단한지 그리고 2021~2022년에 React JS를 배우게 되는 것이 얼마나 엄청난 건지 알게 될 것입니다.
